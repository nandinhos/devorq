# v2.0.0 — Skill de Code Review (Auditoria Técnica)

## Visão Geral
Esta versão transforma a skill `code-review` em um motor de auditoria técnica profunda para o ecossistema DEVORQ, focando em segurança, escalabilidade e arquitetura limpa.

## Mudanças Principais
- **Faseamento de Auditoria**: Processo estruturado em 7 fases (Reconhecimento, Segurança, Arquitetura, Qualidade, Performance, Cobertura, Incoerências).
- **Stack Aware**: Lógica condicional para detectar e auditar Laravel, Node.js, Python e React utilizando referências especializadas.
- **Automação de Shell**: Inclusão de snippets `grep`, `find` e `git` para extração de dados reais do repositório antes da análise qualitativa.
- **Relatório Estruturado**: Saída padronizada com severidade, impacto e solução técnica, além de um Roadmap de Refinamento.
